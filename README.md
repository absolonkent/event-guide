## 📦 Deployment (GitHub Pages)

1. Push code to your GitHub repo.
2. Go to **Settings → Pages**.
3. Choose:
   - Source: `main` branch
   - Folder: `/ (root)`
4. Save and access your live site via the GitHub Pages URL.
